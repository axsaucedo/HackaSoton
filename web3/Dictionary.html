Technical Dictionary

If you have found any terms or phrases used throughout our website and videos confusing, here is a dictionary to explain what these terms mean.

“bunch of bits” - refers to users data only being relevant as binary information.

“The Semantic Web” - led by the World Wide Web Consortium (W3C), coined by Sir Tim Berners-Lee, the Semantic Web is set to provide a common framework that allows for data to be shared and reused across applications.

“revolution” - a forcible overthrow of a previous system (Web 2.0)

“evolving” - where something progresses over time from simple to something more complex.

“intelligent search and behavioural advertisements” - more specific and predictive search engine functionality. More specific advertisements which take into account user interactions - creating more effective campaigns for businesses.

“personalisable” - users can alter and change viewable data and its layout.

“content is generated” - data is created on the web.

“content management systems” - a program that allows for the publishing, editing and modifying of data as well as maintenance from a central interface.

“RSS” - Rich Site Summary is a group of web feed formats used to publish regular updates.

“AJAX” - a group of web development techniques utilised on the client-side to create web applications.

“XML” - Extensible Markup Language, defines a set of rules for encoding documents, formatting them to be both human and machine-readable.

“augmented reality” - a live (direct or indirect) view of realities environments which have been altered with computer generated features.

“advanced mobile technology” - focuses on biometrics (using individuals physical features to secure technology).

“cloud computing” - the application and resources that are accessed using hardware and software over a network.

“retina display” - a brand name used by Apple for liquid crystal displays, claimed to have a high enough pixel density that the human eye can’t notice pixelation, within a standard viewing distance.

“django” - used to ease the creation of complex, database-driven websites.

“ruby on rails“ - an open source full-stack web application framework for the Ruby programming language.

“hadoop” - an open-source software framework that supports data-intensive distributed applications.
